> ### Git 작업영역

- Working Directory (Local)
  + 개인 코드 작성
- Staging (Virtual)
  + git add 를 통해서 수정된 코드를 올리는 영역
- Repository (GitHub)
  + git commit 을 통해서 최종 수정본을 제출
