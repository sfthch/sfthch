SOURCE SYSTEM : 개발DB
DESTINATION(TARGET) SYSTEM :  운영DB
이관대상 테이블 : tbmsd009
상세조건(쿼리)
대상 : SELECT * FROM tbmsd009 WHERE AFFI_CLSS_CD IN ('01','03')
건수
AFFI_CLSS_CD = '01' => 21건
AFFI_CLSS_CD = '03' => 42건
이관사유 : 해당 데이터는 통합 모빌리티 FAQ,  차량공유FAQ 데이터로써 제휴사와 연계를 통해 데이터를 적재하지 않고, 직접 생성하거나 오프라인으로 IMPORT 를 하는 데이터이기에 개발데이터를 운영으로 이관요청 드립니다. 참고로 개발 db에는 AFFI_CLSS_CD = '02' 인 나인투원 데이터도 있지만, 이는 운영에서 연계를 통해 생성될 것 이므로 이관대상에서 제외시켰습니다.
